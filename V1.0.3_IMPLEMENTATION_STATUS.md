### Phase 2: 连接池、快捷键、Avro/Protobuf、高级搜索

#### Avro/Protobuf 消息格式支持 ✅
**文件**:
- `src/utils/schema/avro.ts` - Avro 格式检测和解析
- `src/utils/schema/protobuf.ts` - Protobuf 格式检测和解析
- `src/components/VirtualMessageList/index.tsx` - 消息渲染

**功能**:

#### 高级搜索语法 ✅
**文件**: `src/utils/search.ts`

**功能**:
- 字段过滤器: `key:value`, `partition:0`
- 排除搜索: `-error`
- 精确短语: `"exact match"`
- 正则表达式: `/pattern/`
- 布尔 AND/OR 支持

#### 快捷键修复记录
**问题**: 快捷键无效（? 和 Ctrl+R 无反应）
**原因**: KeyboardShortcutsHelp 组件和 useKeyboardShortcuts hook 没有被挂载到应用中
**修复**:
1. 在 MainLayout 中引入并渲染 KeyboardShortcutsHelp 组件
2. 在 MainLayout 中使用 useKeyboardShortcuts hook 注册全局快捷键
3. 在 TopicListPage 中添加对 'kafkit:refresh' 事件的监听
4. 更新 useKeyboardShortcuts hook 排除 select 元素并添加调试日志

#### 新增功能：Ctrl+K 快速搜索 ✅
**文件**: `src/components/QuickSearch/index.tsx`
**功能**:

- 快速导航（Topic 管理、消费组、设置、连接管理）
- 快速切换连接
- 键盘导航（↑↓ 选择，Enter 确认，Esc 关闭）
- 搜索结果高亮显示

### Phase 3: Schema Registry 支持 ✅

#### Schema Registry 服务
**文件**: `src/services/schemaRegistry.ts`

**功能**:
- 完整的 Confluent Schema Registry API 支持
- LRU 缓存机制 (100 items)
- localStorage 持久化缓存
- 请求去重 (Deduplication)
- 支持 Basic Auth 和 Bearer Token 认证

#### Schema Registry UI
**文件**: `src/components/SchemaRegistryList/index.tsx`

**功能**:
- 多 Registry 配置管理
- 连接测试（添加前验证）
- Subject 列表浏览
- Schema 版本查看
- 类型标签 (Avro/Protobuf/JSON)
- 删除确认

#### 消息解析集成

**功能**:
