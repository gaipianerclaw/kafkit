Avro/Protobuf 格式测试

点击按钮复制测试数据到剪贴板，然后在 Kafkit Producer 中粘贴发送

1. Avro JSON 格式

{
  "__avro_schema": "user.v1",
  "name": "Test User",
  "email": "test@example.com",
  "age": 30,
  "address": {
    "street": "123 Main St",
    "city": "New York"
  }
}

2. Protobuf JSON 格式

{
  "@type": "type.googleapis.com/Order",
  "orderId": "ORD-12345",
  "amount": 99.99,
  "items": [
    {"id": "ITEM-1", "name": "Product A"},
    {"id": "ITEM-2", "name": "Product B"}
  ]
}

3. 批量测试数据

（每行一条，可在 Producer 批量发送模式中使用）

验证步骤

  1. 点击按钮复制测试数据
  2. 打开 Kafkit 应用
  3. 选择一个 Topic，点击"发送消息"
  4. 选择"单条"或"批量"模式
  5. 粘贴测试数据并发送
  6. 切换到"消费消息"查看展示效果

预期效果

  • Avro 消息：展开后显示 [Avro] 橙色标签
  • Protobuf 消息：展开后显示 [Protobuf JSON] 蓝色标签
  • JSON 内容应正确格式化和高亮显示
